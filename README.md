# Robust-Control-of-a-TWIP
Robust control of a Two-Wheeled Inverted Pendulum (TWIP). Features H-infinity loop-shaping, Youla-Kucera parameterization, Mu-analysis for robust stability validation, and controller order reduction for resource-constrained hardware.
